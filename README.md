# 🌐 Web Technology Lab

A collection of web technology lab exercises covering HTML, CSS, and JavaScript fundamentals.

## 📚 Labs Overview

| Lab | Topic | Description |
|-----|-------|-------------|
| Lab 1 | Basic HTML | Page layout with semantic elements |
| Lab 2 | CSS Styling | Basic CSS styling and selectors |
| Lab 3 | HTML Elements | Various HTML elements and structure |
| Lab 4 | CSS Layout | CSS layout techniques |
| Lab 5 | Advanced CSS | Advanced styling and animations |
| Lab 6 | Responsive Design | Responsive web design |
| Lab 7 | Flexbox | Flexbox layout implementation |
| Lab 8 | JavaScript Basics | Introduction to JavaScript |
| Lab 9 | DOM Manipulation | Document Object Model |
| Lab 10 | Advanced JS | Advanced JavaScript concepts |

## 🚀 Getting Started

1. Clone the repository
2. Open any lab folder (e.g., `lab1/`)
3. Open `index.html` in your browser

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
